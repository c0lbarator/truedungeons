Как запустить?
1. Создать виртуальное окружение и установить зависимости:
pip install -r requirements.txt
2.  Создать файл .env (заполнить example.env и переименовать в .env)
3. Создать БД, запустив db.py
3. Запустить bot.py
--------------
Как начать игру?
1. Запустить бота командой /start
2. Следовать инструкциям бота (также можно ознакомиться с туториалом: https://teletype.in/@retrobich921/N1oy7GOnpku#4Cy2 )
